About

Personal Local Weather is a Web application that provides information on weather forecasts. This information varies depending on the location of the user. The features of the application are:

  • An icon followed by a summary;
  • Current temperature in Fahrenheit followed by a button that changes the temperature unit;
  • The wind speed in miles per hour followed by a button that changes the speed unit;
  • The user's current city, state (or equivalent), and country; and
  • A background image that changes according to the weather.

This application wouldn't be made without the help of:

  • free Code Camp;
  • Mozilla Developer Network;
  • Dark Sky;
  • Google Maps APIs;
  • crossorigin.me;
  • CORS Anywhere;
  • Google Fonts;
  • Font Awsome; and
  • Bootstrap.
